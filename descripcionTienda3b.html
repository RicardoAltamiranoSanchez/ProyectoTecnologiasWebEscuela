Este proyecto se utilizo varias tecnologias para el enmaquetado se realizo html css y javascript Con una animacion de particulas para ser mas atractivo el diseño con la pagina principal con un login que tiene un token que se hizo con json para verficar el correo se a del usuario la conexion se hizo con un orm de sqllite para la conexion de la base datos con postgesql y utilizamos a heroku para alogar nuestra aplicacion la conexion se dio atravers de python y flask teniendo todo el control en la parte del backend con un manejo de cookies para automizar la aplicacion y que se mas rapido en navegar y encryptacion de 2ª tipo.Nuestra base de datos esta alojada en heroku teniendo un controlador en postgress donde hacemos las actualizaciones y cambios.
