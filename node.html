Administrador de Usuarios y Productos

Es un proyecto donde escolar donde utilize react js para ala aplicacion de adminstracion donde se pueda adminstrar permiso y privilegios de usuarios y la creacion de categorias y productos

Donde hice un servidor con noje js para ejecutar todos el procedimiento donde esta conectado con mongo atlas para el servidor de hosting y donde se puede adminstrar la apliacion desde la sitaxis de mongo

Tiene token para autenticacion se debe de crear un usuario para poder utilizar la aplicacion
